# Code Of Conduct

JarvisNano follows a simple standard: be direct, respectful, and useful.

## Expected Behavior

- Keep discussions technical and specific.
- Assume contributors are here to improve the project.
- Give actionable feedback with file paths, logs, commands, or screenshots when
  possible.
- Respect that hardware bring-up is messy; document failures without blame.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language
- Publishing another person's private information
- Sharing secrets, credentials, private logs, or private local paths
- Repeated off-topic or bad-faith behavior after maintainers redirect the
  discussion

## Enforcement

Maintainers may edit, hide, or delete comments; close issues; block users; or
otherwise moderate project spaces to keep the project productive and safe.
