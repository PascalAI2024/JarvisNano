Install JarvisNano firmware

Flashes the merged 7.6 MB firmware bundle (bootloader + partition table + OTA data + app + storage) directly to your Seeed XIAO ESP32-S3 Sense over WebSerial. No CLI, no esptool — just a USB-C cable.

  1. Hold the BOOT button on the XIAO.
  2. Plug the device in via USB-C while still holding BOOT.
  3. Release BOOT.
  4. Click Connect below, pick the /dev/cu.usbmodem* port (or COM* on Windows), and let it install.
  5. When done, the device will reboot and broadcast esp-claw-XXXXXX as a Wi-Fi AP. Connect, browse to onboard.html, and run through setup.
Browser support: Web Serial only works on Chromium browsers (Chrome, Edge, Brave, Arc, Opera). Safari and Firefox do not support Web Serial — switch browsers if the install button is greyed out.
Button unavailable? Download jarvis-xiao-esp32s3-sense.bin and flash with esptool.py --chip esp32s3 write_flash 0 jarvis-xiao-esp32s3-sense.bin.
