# Support

JarvisNano is an open-source hardware project. The fastest way to get help is
to include exact hardware and logs.

## Before Asking

- Read [docs/BUILD.md](docs/BUILD.md) for build and flash setup.
- Read [docs/HARDWARE.md](docs/HARDWARE.md) before wiring power, speaker, or
  battery hardware.
- Read [docs/CAMERA.md](docs/CAMERA.md) before debugging the XIAO Sense camera.
- Check [docs/PHASE2_TASKS.md](docs/PHASE2_TASKS.md) for known blockers.

## Good Bug Reports Include

- Board model and purchase batch if known
- Host OS
- Flash command used
- Serial boot log from reset through failure
- Dashboard browser and URL
- Android phone model and Android version, for companion app issues

Do not paste API keys, Wi-Fi passwords, bot tokens, or private local paths.
